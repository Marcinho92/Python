#readme

Repozytorium skladajace sie z dwoch plikow:
a) Flight.py zawierajcy klasy Plane oraz Simulate wykorzystywane do symulacji lotu samolotu,
b) tests.py zawierajacy testy jednostkowe dla klas Plane oraz Simulate

Aby uruchomic testy nalezy w konsoli uzyc komendy:
python tests.py

co powinno poskutkowac wykonaniem pomyslnie zakoczonych testow.
